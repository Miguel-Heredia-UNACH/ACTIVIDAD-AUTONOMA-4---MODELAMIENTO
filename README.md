# Optimización de Algoritmos - Actividad Autónoma 4
Estudiante: Miguel Heredia
Carrera: Ciencia de Datos

Este repositorio contiene la Actividad Autónoma 4 de la Unidad 2. Se incluye un Jupyter Notebook (.ipynb) con la comparación de rendimiento entre un algoritmo ineficiente O(N^2) y uno optimizado usando NumPy y la Criba de Eratóstenes.

Archivos principales:
- codigo_original.py: Notebook con todo el código y gráficas.
- analisis_primos.ipynb: Notebook con todo el código y gráficas.
- DOCUMENTACION.PDF: Informe detallado de los resultados.
- gráfico_tiempos.txt: gráfica obtenida al comparar los dos tiempos.
- profiling_optimizado.txt: Resultados del análisis con cProfile."
